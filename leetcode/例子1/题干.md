**题目**：有 1、2、3、4 四个数字，能组成多少个互不相同且无重复数字的三位数？都是多少？
```rust
fn main() {
    let mut count = 0;
    for i in 1..5 {
        for j in 1..5 {
            for k in 1..5 {
                if i != j && j != k && k != i {
                    println!("{i}, {j}, {k}");
                    count += 1;
                }
            }
        }
    }

    println!("count: {count}");
}
```
**程序分析**：可填在百位、十位、个位的数字都是 1、2、3、4，组成所有的排列后再去掉不满足条件的排列。
